Squeezebox

squeezeBox is a simple modal box jQuery plugin.

View project onGitHub
squeezeBox is a lightweight jQuery Plugin which provides a powerful and great looking transition effect. Plugin provides a cross-browser solution to easily create modal boxes out of any element in your webpage.

USAGE

squeezeBox provides a simple way to invoke a modal dialog.

As a chained jQuery function, you can call the modal() function on a jQuery element and a modal dialog will be displayed using the contents of that element. For example: 

$("#element-id").squeezeBox();

The method described above also accepts an optional options object. Using the example above : 

$("#element-id").squeezeBox({options});

STYLING

Styles can be applied through external CSS, the options object, or both.

squeezeBox handles setting the necessary CSS for displaying the modal dialog. In addition, squeezeBox dynamically centers the modal dialog, unless the position option is used, which takes precedence.

squeezeBox internally defines the following CSS classes: squeezeBox-overlay, squeezeBox-container and squeezeBox-data.

		'padding': '3px'
